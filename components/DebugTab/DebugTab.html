$debug - Debug Tools

Usage

// Logging
$debug.log('User logged in', { userId: 123 })
$debug.warn('Rate limit approaching')
$debug.error('API failed', error)

// Performance
$debug.time('api-request')
await $http('/users')
$debug.timeEnd('api-request') // Shows ms

// Lifecycle tracking
$debug.trackComponent('Hero', 'mount')

// DevTools panel
$debug.panel()

Lifecycle Tracking

Automatically tracks mount, unmount, render, and errors for all SimpleComponent instances. 

$debug.getLifecycle('Hero')
// [{ event: 'mount', timestamp: ..., data: {...} }, ...]

$debug.getLogs()
// [{ type: 'log', data: [...], timestamp: ... }, ...]

Methods

  • $debug.log(label, data) - Log with timestamp
  • $debug.warn(message) - Warning level
  • $debug.error(message, error) - Error level
  • $debug.time(label) - Start timer
  • $debug.timeEnd(label) - End timer, show ms

Dev Mode: $debug auto-enables on localhost or when ?debug in URL.
