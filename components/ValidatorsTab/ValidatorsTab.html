Validators

Usage

Validators.email('user@example.com')  // null = valid
Validators.email('invalid')           // 'Invalid email' = error

// Combine multiple validators
combineValidators(
  Validators.required(),
  Validators.email(),
  Validators.noXSS()
)

Available Validators

email password phone creditCard url ssn zipcode numeric noXSS noSQLi minLength maxLength

Security: Built-in noXSS and noSQLi validators detect common attack patterns.

Custom Validators

const customValidator = (value) => {
  if (value.length < 5) {
    return 'Value must be at least 5 characters'
  }
  return null // null = valid
}

const error = customValidator('hi')
// error = 'Value must be at least 5 characters'
