$debug

Debug tools and lifecycle tracking

Logging

$debug.log('Info message')
$debug.warn('Warning')
$debug.error('Error', err)
$debug.info('Details')

Performance

$debug.time('myTimer')
// ... code ...
$debug.timeEnd('myTimer') // Logs ms

Lifecycle

$debug.trackComponent('Hero', 'mount')
$debug.getLifecycle('Hero')

DevTools Panel

$debug.panel() // View all logs
$debug.exportLogs()
$debug.clearLogs()

Auto-enabled

$debug auto-enables on localhost or when ?debug in URL. Automatically tracks component lifecycle (mount, unmount, render, errors) for all SimpleComponent instances.
