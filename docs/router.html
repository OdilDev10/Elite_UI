Router

Hash-based routing with guards and middleware

Configuration (pages/router.js)

export default {
    routes: [
        { path: '/', handler: 'HomePage' },
        { path: '/example', handler: 'ExamplePage' },
        { path: '/users/:id', handler: 'UserPage' }
    ],
    guards: [
        (to, from) => {
            if (requiresAuth(to.path) && !isLoggedIn()) {
                router.navigate('/login')
                return false
            }
            return true
        }
    ],
    middleware: [
        (path) => $debug.log('Navigating to:', path)
    ]
}

Navigate

router.navigate('/users/123')
router.back()
router.forward()

Query Params

router.getQueryParams() // { page: '1' }
router.setQueryParams({ page: 2 })
router.updateQueryParam('sort', 'date')

Features

Hash-based (default) or HTML5 history • Path parameters • Query params • Navigation guards • Middleware • XSS sanitization on all params
