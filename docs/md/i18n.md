# Sistema i18n - Internacionalización

## Overview

El sistema i18n de EliteUI permite manejar múltiples idiomas de forma sencilla con archivos JSON.

## Inicialización

```javascript
// Inicializar con idioma por defecto
initI18n('es')

// Ahora $t está disponible globalmente
$t('home.title')
```

---

## Cargar Traducciones

### Desde objeto JSON directo

```javascript
$i18n.load({
    en: {
        home: {
            title: 'Welcome',
            subtitle: 'Hello world'
        },
        nav: {
            home: 'Home',
            about: 'About'
        }
    },
    es: {
        home: {
            title: 'Bienvenido',
            subtitle: 'Hola mundo'
        },
        nav: {
            home: 'Inicio',
            about: 'Acerca de'
        }
    }
})
```

### Desde archivos externos

```javascript
// Cargar desde archivos JSON
await $i18n.loadFromFiles('lang/', ['en', 'es', 'pt'])

// Busca: lang/en.json, lang/es.json, lang/pt.json
```

## Uso en JavaScript

### Función $t

```javascript
// Traducción simple
$t('home.title')  // 'Bienvenido'

// Con parámetros
$t('greeting', { name: 'Juan' })
// Traducción: 'Hola {{name}}' → 'Hola Juan'
```

### Cambiar idioma

```javascript
// Cambiar idioma actual
$i18n.setLocale('en')

// Obtener idioma actual
$i18n.getLocale()  // 'en'

// Obtener todas las traducciones del idioma actual
$i18n.getTranslations()
```

## Uso en HTML

### Directivas data-*

```html
<!-- Con SimpleComponent -->
<span data-text="title"></span>

<!-- En el componente -->
class HomePage extends SimpleComponent {
    render() {
        this.setState({
            title: $t('home.title')
        })
        this.el.innerHTML = `<h1 data-text="title"></h1>`
    }
}
```

### Cambio dinámico de idioma

```javascript
// Cuando el usuario cambia el idioma
function changeLanguage(lang) {
    $i18n.setLocale(lang)
    
    // Actualizar todos los componentes que listen cambios
    window.dispatchEvent(new CustomEvent('i18n:changed', { detail: lang }))
}

// En componentes
window.addEventListener('i18n:changed', () => {
    this.setState({ /* nuevas traducciones */ })
    this.render()
})
```

## Ejemplo de Página con i18n

### HomePage.js

```javascript
class HomePage extends SimpleComponent {
    constructor(selector) {
        super(selector, {
            title: $t('home.title'),
            subtitle: $t('home.subtitle'),
            navHome: $t('nav.home'),
            navAbout: $t('nav.about')
        })
    }

    render() {
        this.el.innerHTML = `
            <h1 data-text="title"></h1>
            <p data-text="subtitle"></p>
            <nav>
                <a href="#/" data-text="navHome"></a>
                <a href="#/about" data-text="navAbout"></a>
            </nav>
        `
    }
}
```

## Cambio de Idioma desde UI

```javascript
// Selector de idioma en el header
class LanguageSelector extends SimpleComponent {
    constructor(selector) {
        super(selector, {
            currentLang: $i18n.getLocale()
        })
    }

    render() {
        this.el.innerHTML = `
            <select data-onchange="changeLanguage" class="px-2 py-1 border rounded">
                <option value="en" data-selected="currentLang === 'en'">English</option>
                <option value="es" data-selected="currentLang === 'es'">Español</option>
                <option value="pt" data-selected="currentLang === 'pt'">Português</option>
            </select>
        `
    }

    changeLanguage(e) {
        const lang = e.target.value
        $i18n.setLocale(lang)
        localStorage.setItem('elite-locale', lang)
        
        // Notificar a la app
        window.dispatchEvent(new CustomEvent('i18n:changed', { detail: lang }))
    }
}
```

## Detectar Idioma del Navegador

```javascript
// Al inicializar
function initAppLanguage() {
    // 1. Check localStorage
    const saved = localStorage.getItem('elite-locale')
    if (saved) return saved

    // 2. Check navegador
    const browserLang = navigator.language.split('-')[0]
    const supported = ['en', 'es', 'pt']
    if (supported.includes(browserLang)) {
        return browserLang
    }

    // 3. Default
    return 'en'
}

// En bootstrap
initI18n(initAppLanguage())
```

## API Reference

| Método | Descripción |
|--------|-------------|
| `initI18n(locale)` | Inicializa con idioma por defecto |
| `load(data)` | Carga traducciones desde objeto |
| `loadFromFiles(path, locales)` | Carga desde archivos JSON |
| `setLocale(locale)` | Cambia idioma actual |
| `getLocale()` | Obtiene idioma actual |
| `t(key, params)` | Traduce una key |
| `getTranslations()` | Obtiene todas las traducciones |
