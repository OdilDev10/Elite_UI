Validators

Security & input validation built-ins

Usage

Validators.email('test@mail.com')  // null = valid
Validators.email('invalid')         // 'Invalid email' = error

Validators.required()('value')     // null
Validators.required()('')          // 'This field is required'

Combine

combineValidators(
    Validators.required(),
    Validators.email(),
    Validators.noXSS()
)

Format

email, phone, creditCard, url, ssn, zipcode

Security

noXSS, noSQLi, pattern, custom

Password

min, uppercase, numbers, special chars
